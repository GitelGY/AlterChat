# 🎭 AlterChat - הבוט עם אינסוף פנים

**AlterChat** הוא צ'אט-בוט אינטליגנטי וגמיש המבוסס על מודל ה-AI של גוגל (Gemini), המאפשר לך להגדיר לו "פרסונה" (תפקיד) וסגנון דיבור ייחודיים בכל שיחה מחדש.

## 🌟 למה AlterChat?
בניגוד לצ'אט-בוטים רגילים, **AlterChat** מציע חופש פעולה מלא:
* **התאמה אישית:** הגדירי לבוט "מי הוא" (מורה, מתכנת, מאמן כושר או חבר תומך) באמצעות ממשק פשוט.
* **גמישות:** אפשרות לשנות את הפרסונה תוך כדי תנועה בשיחה.
* **זיכרון שיחה:** הבוט זוכר את הוראות התפקיד ואת רצף ההודעות.
* **עיצוב מודרני:** ממשק משתמש כהה, נקי ויוקרתי המבוסס על Streamlit.

## 🚀 טכנולוגיות
- **Streamlit:** ממשק Web מהיר ואינטראקטיבי.
- **Google Gemini API:** מנוע הבינה המלאכותית המתוחכם.
- **Python:** השפה שמאחורי הקלעים.
